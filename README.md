# .

To install dependencies:

```bash
bun install
```

To create database:
1. create .env and set user and password
2.
```bash
bunx prisma migrate dev
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.8. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
